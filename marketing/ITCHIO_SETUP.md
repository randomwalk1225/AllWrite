# itch.io 페이지 설정 가이드

## 1. 계정 생성

1. https://itch.io 접속
2. "Register" 클릭
3. 이메일, 사용자명, 비밀번호 입력
4. 이메일 인증

## 2. 새 프로젝트 생성

1. 우측 상단 프로필 → "Dashboard"
2. "Create new project" 클릭
3. "Other" 카테고리 선택 (교육용 소프트웨어)

## 3. 기본 정보 입력

### Title
```
AllWrite 2.0 - Interactive Math Geometry Tool
```

### Short description (추천)
```
Create geometric shapes with real-time synchronization. Perfect for math education - drag points and watch shapes transform instantly.
```

### Classification
- **Kind of project**: Tool
- **Release status**: Released
- **Pricing**: Free 또는 Pay what you want

### 가격 설정 옵션

#### Option A: 완전 무료
- "No payments" 선택
- 가장 많은 다운로드

#### Option B: Pay What You Want (추천)
- "No minimum" 선택
- "Suggested donation" 입력: $5
- 자발적 후원 유도

#### Option C: 최소 가격 설정
- "Minimum price" 설정: $3-5
- 수익화 가능하지만 다운로드 감소

## 4. 상세 설명 (Description)

```markdown
# AllWrite 2.0

**Interactive geometry and graphing tool for math education**

## Features

### Geometry Tools
- Points (fixed, midpoint, division point, constrained)
- Lines (segment, ray, infinite line)
- Circles (center-radius, diameter, three-point)
- Polygons (regular 3-12 sided, special shapes, free-form)

### Real-time Synchronization
The core feature of AllWrite 2.0: **drag any point and watch all connected shapes transform in real-time**.

- Drag polygon center → entire shape moves
- Drag radius point → resize and rotate
- Drag line endpoint → line follows instantly

### Drawing & Canvas
- Infinite canvas with pan/zoom
- Natural pen strokes (Krita-style brush engine)
- Eraser tool
- Selection tools (click, rectangle, lasso)

### Export
- Save/Load projects
- Export to PDF
- Export to image

## Perfect For
- Math students (middle school ~ high school)
- Math teachers and tutors
- Educational institutions
- Self-learners exploring geometry

## System Requirements
- Windows 10/11 or macOS 11+
- 4GB RAM
- 200MB disk space

## License
MIT License - Free and open source

## Links
- [GitHub Repository](https://github.com/randomwalk1225/AllWrite)
- [Report Issues](https://github.com/randomwalk1225/AllWrite/issues)
```

## 5. 파일 업로드

### 업로드할 파일
1. Windows 설치 파일: `AllWrite2-Setup-2.0.0.exe`
2. macOS DMG: `AllWrite2-2.0.0.dmg`
3. (선택) ZIP 버전

### 플랫폼 태그 설정
- Windows 파일 → "Windows" 체크
- macOS 파일 → "macOS" 체크

## 6. 이미지 업로드

### Cover Image (필수)
- 크기: 630x500 픽셀 권장
- 내용: 로고 + 주요 기능 스크린샷

### Screenshots (최소 3개)
1. 메인 화면 전체
2. 기하학 도형 예시
3. 실시간 동기화 데모 (GIF 가능)

### GIF 가능
- itch.io는 GIF 업로드 지원
- 동적인 기능 시연에 효과적

## 8. 추가 설정

### Community
- "Enable comments" 체크 (피드백 수집)

### Analytics
- "Enable Google Analytics" (선택)
- 방문자 통계 확인 가능

### Visibility
- "Draft" → 미리보기 확인
- "Public" → 공개 (검색 가능)
- "Restricted" → 링크 가진 사람만
