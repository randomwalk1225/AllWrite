## 필수 스크린샷

### 1. 메인 화면 (main.png)
- [ ] 크기: 1920x1080 (Full HD)
- [ ] 내용: 여러 도형이 그려진 상태의 전체 화면
- [ ] 포함 요소: 왼쪽 도구 패널, 캔버스에 도형들, 좌표 평면

### 2. 기능별 스크린샷

#### geometry.png - 기하학 도구
- [ ] 크기: 1280x720 이상
- [ ] 내용: 다양한 도형 (정다각형, 원, 선분) 조합
- [ ] 강조: 점과 도형의 연결 관계

#### pen.png - 펜 드로잉
- [ ] 크기: 1280x720 이상
- [ ] 내용: 자연스러운 펜 스트로크 예시
- [ ] 강조: 다양한 색상, 굵기

#### graph.png - 그래프 기능
- [ ] 크기: 1280x720 이상
- [ ] 내용: 수식과 그래프
- [ ] 강조: 좌표 평면, 함수 그래프

### 3. 데모 GIF (중요!)

#### polygon-demo.gif
- [ ] 길이: 5-10초
- [ ] 내용: 정육각형 그리기 → 점 드래그로 변형
- [ ] 파일 크기: 5MB 이하 권장

#### circle-demo.gif
- [ ] 길이: 5-8초
- [ ] 내용: 원 그리기 → 크기 조절

#### midpoint-demo.gif
- [ ] 길이: 8-12초
- [ ] 내용: 두 점 생성 → 중점 생성 → 원래 점 이동 시 중점 따라감

#### pen-demo.gif
- [ ] 길이: 5-8초
- [ ] 내용: 펜으로 자연스럽게 그리기

### 4. 튜토리얼 이미지

#### step1.png - 도구 선택
- [ ] 크기: 800x500 이상
- [ ] 내용: 도구 패널 확대 + 선택 하이라이트

#### step2.png - 도형 작도
- [ ] 크기: 800x500 이상
- [ ] 내용: 캔버스에서 도형 그리는 중

#### step3.gif - 실시간 조작
- [ ] 길이: 5-8초
- [ ] 내용: 점 드래그 → 도형 변형

## GIF 제작 도구

### Windows
1. **ScreenToGif** (무료, 추천)
   - https://www.screentogif.com/
   - 화면 녹화 + GIF 편집

2. **ShareX** (무료)
   - https://getsharex.com/
   - 스크린샷 + GIF 녹화

### 온라인
1. **ezgif.com**
   - 비디오 → GIF 변환
   - GIF 최적화/압축

## 이미지 최적화

### PNG 압축
- TinyPNG: https://tinypng.com/
- Squoosh: https://squoosh.app/

### GIF 최적화
- ezgif Optimizer: https://ezgif.com/optimize
- 목표: 5MB 이하, 15fps

### 권장 설정
- 스크린샷: PNG, 품질 80-90%
- GIF: 15fps, 720p 해상도
- 소셜 미디어용: 1200x630 (OG 이미지)

---

## 촬영 팁

1. **배경 정리**: 불필요한 창/아이콘 제거
2. **줌 레벨**: 100% 또는 적절한 확대
3. **색상**: 밝고 대비가 좋은 색상 사용
4. **해상도**: 최소 720p, 가능하면 1080p
5. **GIF 속도**: 너무 빠르지 않게 (실제 사용 속도)
